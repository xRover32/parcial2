  • El position fixed este igual esta utilizando marcos de referencia, con la diferencia que el elemento se muestra en una posición fija
  • El position absolute
  • El posicionamiento sticky se suele utilizar cuando queremos que un elemento se posicione en un lugar específico de forma fija

Display

Ideas Principales

  • Por defecto, todos los elementos (div) se comportan como elementos de bloque (block)
  • Los elemntos inline se colocan a continuacion del otro
  • El elemento inline-block se comporta como inline pero no ignora width o height.
  • El elemento flexutiliza el modelo de cajas flexibles de CSS
  • El elemneto inline-flexes como un elemento en línea y se establece su contenido de acuerdo con el modelo flexbox
  • El elemnto grid es de bloque y establece su contenido de acuerdo a cuadrícula o rejillas
  • El elemnto inline-grid utiliza solo el contenido de las cajas del grid
  • El elemento none oculta los elementos pero sigue existiendo

Animation

Ideas Principales

  • animation-duration especifica cuanto tiempo debe transcurrir una animacion para completar un ciclo
  • animation-iteration especifica el nuemro de veces que se debe reproducir una animacion
  • animation-delay especifica un restraso en la animacion
  • animation-play-state especifica si es el pause/start de la animacion
  • animation-timing-function especifica la curva de la velocidad de la animacion
    • ease:empieza lento, luego rapido y termina lento (por defecto)
    • ease-in:especifica una animacion con un inicio lento
    • ease-out:especifica una animacionn con un final lento
    • ease-in-outespecifica una animacion con inicio y final lento
    • cubic-bezier(n,n,n,n):permite definir sus propios valores
  • animation-direction especifica si una animacion debe de reproducirse hacia adelante, atras o en ciclos
    • normal:La animacion empieza normal
    • reverse:Empieza de reversa
    • alternate:La animacion va primero de adelante y luego hacia atras
    • alternate-reverse:La animacion empieza hacia tras y luego hacia adelante

Apuntes Tomados

  • CSS permite animar la mayoria de los elemntos sin usar JavaScript
  • Los @keyframes dentro de la reglas cambiara el estilo a uno nuevo
  • Siempre se deben de vincular los elementos para que la animacion se haga
  • animation-duration
  • animation-iteration
  • animation-delay
  • animation-play-state
  • animation-timing-function:linear, ease, ease-in, ease-out, ease-in-out y cubic-bezier(n,n,n,n)
  • animation-direction:normal, reverse, alternate, alternate-reverse

Resumen

En esta clase vimos lo que es la animacion y como se le puede agregar a los objetos y esto permita que las paginas web sean mas dinamicas y tengan su toque definiendo los tipos de animacion y orientacion de las animaciones que queramos ponerle a nuestra pagina web

Nuevas Propiedades CSS3

Ideas Principales

  • La propiedad Box-shadow crea sombras en la caja mecionada y este puede formar 5 parametros de color en difentes direcciones y difuminaciones
  • La propiedad @font-face permite utilizar y usar cualquier fuente que se necesite
  • La propiedad linear-gradient genera una gradiente lineal em los background
  • La propiedad HSLA son el tono,saturacion y luminosidad
  • La propiead Outline con otra propiead llamada outline-offset generan un segundo borde alejado del borde original de cierto elemento
  • La propiedad Border-image crea un borde con una imagen personalizada y se necesita que borde sea declarado con las propieades borde o border-with
  • La propiedad Transform modifica la forma de un elemento usando scale, rotate, skew y translate

Apuntes Tomados

  • CSS3 cubre diseño y estilos web junto con formas y movimiento
  • Pueden ser trasnformaciones, esquinados, reposicionamiento y sombras
